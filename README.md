## 개발 내용
#### 개발 기간: 2024.07 - 2024.-08
#### 사용한 기술 스택
  - **코어**: Next.js, TypeScript
  - **스타일링 및 애니메이션**: Tailwind CSS, GSAP
  - **3D 그래픽**: Three.js (R3F)
  - **패키지 매니저**: Bun
  - **빌드 도구**: SWC
  - **배포**: Vercel
  - **에러 추적 및 분석**: Sentry, GA4
    
#### 브랜치 전략
  - Git Flow 전략을 기반으로 `main`, `develop` 브랜치 운용
  - 기능 개발은 `develop` 브랜치에서 진행하며, 충분히 테스트된 후 `main` 브랜치로 병합하여 배포
